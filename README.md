# HomeBanking

Implementación de un pseudo-homebanking.

Tecnologías/liberías utilizadas hasta el momento:
Nodejs, Express, Mongodb, Mongoose, Chai, Mocha, Supertest, Nodemon, Mockgoose, Morgan, Jshint.

Comandos:
*jshint src 
*npm run start
*npm run test
